Proyectos: Detalle de Interfaces

Módulos construidos para Innova Negocios.

Laravel/PHP SQL Server BPMN/Kanban

Módulo de Transacciones y Ventas

Automatización completa del proceso de venta. Permite el registro rápido de transacciones, la asociación de clientes y la gestión del carrito de compras.

  • Generación automática de boletas (RF06).
  • Validación de stock en tiempo real antes de completar la venta (RF04).
  • Historial de ventas con trazabilidad de pagos.

Arquitectura de Datos y Stock

Diseño de la base de datos relacional en SQL Server (Figura 12) para soportar la integridad de inventario y la escalabilidad del sistema.

  • Diseño de Tablas de Productos, Ventas y Usuarios.
  • Reporte detallado de Inventario con opción a exportación (RF07).
  • Estructura preparada para Procedimientos Almacenados y Triggers.

Modelado de Procesos (BPMN)

Aplicación de metodologías de ingeniería para diseñar el flujo de trabajo antes de la codificación, asegurando un proceso lógico y optimizado.

  • Diagramas de Flujo (Flujo de Acceso, Registro de Ventas).
  • Control de Acceso mediante roles definidos (RNF01).
  • Cumplimiento del Requerimiento No Funcional (RNF04) de Tiempo de Respuesta < 2s.

Dashboard Administrativo

Interfaz intuitiva que ofrece reportes detallados y resumen de inventario para facilitar la toma de decisiones estratégicas.

  • Visualización de Ventas Totales y Métrica de Stock.
  • Base de datos de Clientes con historial de compra.
  • Generación de Reporte de Inventario en formato PDF/Excel (RF07).

Catálogo de Productos & Frontend

Diseño de la interfaz de usuario de cara al cliente con funciones completas de e-commerce. Muestra mis habilidades de desarrollo frontal (Frontend) y lógica de negocio.

  • Sistema de filtrado por Categorías y Precio (UX/UI).
  • Funcionalidad de Carrito de Compras que persiste en Local Storage.
  • Proceso de compra con Login de Cliente.

Herramientas de Gestión Interna

Vista ampliada de las herramientas de gestión interna. Este panel centraliza el control de inventario, el estado de las ventas y la administración de usuarios y categorías.

  • Manejo de Productos (14), Ventas (13) y Clientes (5).
  • Capacidad para crear, editar y eliminar registros (CRUD).
  • Diseño responsive para uso administrativo.

Login y Autenticación de Empleados

Implementación de un sistema de autenticación robusto para personal interno, asegurando que solo usuarios autorizados puedan acceder a los módulos de gestión y reportes.

  • Autenticación segura basada en tokens/sesiones de Laravel.
  • Separación estricta de vistas Admin y vistas Cliente.
  • Protección de rutas críticas mediante Middleware.
